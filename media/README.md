# Vidéo de démonstration

Cette vidéo montre le **fonctionnement réel du projet** de contrôle PID de l’humidité du sol.
**Nom du fichier** : https://youtube.com/shorts/U-skX34zJJk?feature=share

Elle prouve que le système est **réalisé et opérationnel** :
- lecture de l’humidité du sol,
- régulation automatique par PID,
- activation et arrêt de la pompe via Arduino.

La vidéo sert de **preuve de faisabilité et de bon fonctionnement** du projet.
